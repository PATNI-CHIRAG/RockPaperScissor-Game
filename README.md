# ✊✋✌️ Rock Paper Scissors Game

![Rock Paper Scissors Screenshot](./Screenshot%202026-05-26%20142720.png)

A fun and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

---

## 🚀 Features

✅ Player vs Computer Gameplay  
✅ Live Score Tracking  
✅ Random Computer Moves  
✅ Round Counter  
✅ Winner Detection  
✅ Auto Reset After 5 Wins  
✅ Interactive Hover Effects  
✅ Colorful UI Design  

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 Game Rules

- Rock beats Scissors ✊
- Paper beats Rock ✋
- Scissors beats Paper ✌️
- First player to reach 5 points wins the game.

---

## 🧠 JavaScript Functions

### `compChoose()`

- Generates random computer choice

### `winner(compChoice, youChoice)`

- Checks winner of the round
- Updates scores
- Displays result message

### `reset(compWin, youWin)`

- Checks if any player reached 5 points
- Declares final winner

### `setZero()`

- Resets scores and rounds
